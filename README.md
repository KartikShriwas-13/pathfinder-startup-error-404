# Startup Launchpad AI 🚀

An AI-powered startup idea validation platform that helps entrepreneurs analyze business ideas, identify risks, and generate actionable insights.

## 🔥 Features

- 🔐 User Authentication (Supabase)
- 💡 AI Startup Idea Analysis
- 📊 Dashboard with Idea History
- 📈 Profitability & Risk Insights
- 📍 Location-based Suggestions
- 🧠 AI-generated Business Roadmap

## 🛠 Tech Stack

- React + TypeScript
- Vite
- Tailwind CSS + shadcn UI
- Supabase (Auth + Database + Functions)
- TanStack React Query
